# Shape Constraint Language (SHACL)
[SHACL](https://en.wikipedia.org/wiki/SHACL){target="_blank" rel="noopener noreferrer"} is a language to define constraints for graph data. The [linkml Shacl Exporter](https://linkml.io/linkml/generators/shacl.html){target="_blank" rel="noopener noreferrer"} can export the data tests from the schema files to SHACL files.
